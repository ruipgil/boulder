boulder
=======

Simple static server starter.

To start a static server type

``` boulder ```

simple, and fast.

More options
------

Use ```-p``` or ```--port``` to pass a specific port.

Put more arguments to serve more folder.
Folders with the same name are not supported.

Help with ```-h``` or ```--help```.

Install
------

``` npm install boulder -g ```
